## 📅 진행 방식

### ✅ **\[평일 매일 1문제 풀이]**

* 백준 티어 기준으로 문제를 분배합니다:

  * 브론즈: 5문제
  * 실버 3\~5: 각 3문제
  * 실버 1\~2: 각 2문제
  * 골드: 1문제
  * 플래티넘: 1문제당 2일 소요

* 각자 자신의 디렉토리에 풀이 코드를 업로드합니다.
